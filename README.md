### About
ClassCharts Improver adds many new features to ClassCharts, including:

- Cleaner design
   - New feather icons
   - Cleaner card design (rounded corners)
- Access personal notes
- Set Personal Goals
- Set a Custom Profile Photo from the side menu.


### Using the extension

#### Option 1:
Download from [Chrome Web Store](https://chromewebstore.google.com/detail/classcharts-improver/kalmdpfngeebamgaeegkieojhbkbghoe).

#### Option 2 (Download ZIP and host locally using developer mode):
1. Download the latest ZIP version from [here](https://github.com/jamestheakston1/classchartsimprover/releases/).
2. Extract
3. Visit **chrome://extensions** or **edge://extensions** for Microsoft Edge.
4. Toggle developer mode and click 'load unpacked'.
5. Navigate to the folder in the alert and click 'Open'.
6. Use ClassCharts like usual, but with the new features!

#### Option 3 (Clone):
1. Clone using `git clone jamestheakston1/classchartsimprover`
